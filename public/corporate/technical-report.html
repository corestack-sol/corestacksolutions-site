Notas técnicas

// Endpoint problemático: /api/notifications/send
// Causa: await secuencial de FCM tokens — optimizar con Promise.all

// Antes (lento):
for (const token of tokens) {
  await fcm.send({ token, notification });
}

// Después (recomendado):
await Promise.all(tokens.map(token => fcm.send({ token, notification })));
⚠
Acción requerida

El endpoint /api/notifications/send tiene latencia p95 de 820ms. Refactorizar de iteración secuencial a Promise.all reducirá el tiempo estimado a ~180ms.
